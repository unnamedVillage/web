# Unnamed Village Web

This is the SPA for the Unnamed Village.

Uses the Irish Government Design System.

- [Component Guide](https://ds.services.gov.ie/components/setup-guides/react/)
- [Storybook](https://ds.services.gov.ie/storybook-react/)

## Development

1. Install Node 24
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
